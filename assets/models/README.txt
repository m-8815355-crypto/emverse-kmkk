This folder is for 3D model assets.

You currently have the *source code* for the magnet (in the 'src' folder), but the AR viewer needs a compiled 3D file (.glb).

How to get 'magnet.glb':
1. Open 'model-exporter.html' (located in the root folder) in your browser.
2. Click "Generate & Download magnet.glb".
3. Move the downloaded 'magnet.glb' file INTO this folder (assets/models/).
4. Once 'magnet.glb' is here, the AR experience will work!

Files structure should look like:
- assets/models/magnet.glb
- assets/models/src/ (Your code files - keep these here as reference)

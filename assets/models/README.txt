This folder is for 3D model assets.

Please place your 3D model files here:
1. magnet.glb (Required for Android & WebXR)
2. magnet.usdz (Optional, for better iOS Quick Look support)

The 'ar-experience.html' file is configured to look for 'assets/models/magnet.glb'.
If your file has a different name, please update the 'src' attribute in 'ar-experience.html'.
